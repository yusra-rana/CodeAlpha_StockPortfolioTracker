# CodeAlpha Stock Portfolio Tracker 📈

This project is a simple Stock Portfolio Tracker created using Python during the CodeAlpha Python Programming Internship.

## 📌 Features

* User inputs stock names and quantities
* Calculates total investment value
* Uses predefined stock prices
* Beginner-friendly console application
* Optional file saving support

## 🛠 Technologies Used

* Python 3

## 📚 Concepts Used

* Dictionary
* User Input/Output
* Arithmetic Operations
* File Handling

## ▶️ How to Run

1. Install Python on your system
2. Download or clone this repository
3. Run the Python file:

```bash id="8n2wcf"
python stock_tracker.py
```
